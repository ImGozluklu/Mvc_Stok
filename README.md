# Yaz Okulu Kayıt

## Projenin Amacı
- Ürünlerin stoklarını , müşterileri , satışların tutulması. 
- Online şekilde arama yaparak veri bulma.
- Stok kontrolü yapılabilmesi.

## Kullanılan Teknolojiler , Kütüphaneler ve Yöntemler
- HTML - CSS - JavaScript
- ASP.NET MVC
- C#
- Entity Framework
- Bootstrap
- Validation
- Paging
- Popup
